# LPC-Flash (lpcflash)
Linux flashing tool for NXP (Philips) LPC2xxx MCU series (by Iurii Ovcharenko aka amw)

This program allows to read/write/erase/program memory (RAM and Flash)
for Philips LPC2xxx ARM microcontrollers over serial port.

Supported devices:
  * LPC2103
  * LPC2104
  * LPC2105
  * LPC2106
  * LPC2114
  * LPC2119
  * LPC2124
  * LPC2129
  * LPC2131
  * LPC2132
  * LPC2134
  * LPC2136
  * LPC2138
  * LPC2141
  * LPC2142
  * LPC2144
  * LPC2146
  * LPC2148
  * LPC2194
  * LPC2210
  * LPC2212
  * LPC2214
  * LPC2292
  * LPC2294
  * LPC2364
  * LPC2366
  * LPC2368
  * LPC2378
  * LPC2468
